Web semantica

Elementos con significado

Ejemplos de etiquetas semanticas: Table, Form, Article...
Ejemplos de etiquetas no semanticas: Div, Span...

Header

Es el encabezado de una pagina, seccion o articulo. Incluye elementos como el logo, texto introductorio, menus de navegacion. Puede haber multiples headers en una pagina, pero no puede haber un header dentro de un footer o otro header.

footer

Es el pie de pagina de un sitio o seccion. Puede haber multiples por pagina. Puede incluir informacion como:
  • Informacion de contacto
  • Enlaces a la RRSS
  • Mapa del sitio
  • Creditos
  • Formulario de Contacto

Nav

Define un conjunto de enlaces de Navegacion. No todos los links deben estar dentro de un "Nav".

Main

Crea un contenedor para el contenido principal de nuestra pagina. Este debe ser unico y no debe incluir contenido repetido de otras secciones como por ejemplo del sidebar, logod, etc. No puede ser hijo de etiquetas como: article, aside, footer, header o nav.

Aside

Define el contenido relacionado indirectamente con el contenido principal como por ejemplo un sidebar. Elementos que solemos poner en un aside.
  • Publicidad
  • Biografias del autor
  • Enlaces relacionados
  • Informacion de perfil

Section

Define una seccion de una pagina donde se agrupa los contenidos relacionados, y normalmente incluyre un titulo
Una web se compone de varias secciones, cada una con su contenido y proposito
Ejemplos de uso:
  • Capitulos
  • Introduccion
  • Informacion de contacto

Article

Define un contenido INDEPENDIENTE El mismo puede ser compartido o reutilizado en otro sitio sin requerir contexto
Tambien suele incluir un titulo
Ejemplos de uso:
  • Un post de un blog
  • Un articulo de una revista
  • Comentario de un usuario (Resena)
  • Tarjeta de producto

Puedo encontrar articulos dentro de secciones y secciones dentro de articulos. Un articulo puede tener multiples secciones. Una seccion puede tener multiples articulos.

Figure

Define elementos autocontenidos, como puede ser una imagen, un diagrama, un grafico, una tabla o una foto
Dentro de ella, se suele utilizar la etiqueta "figcaption" para describir el contenido, esto es ademas del ya incluido "alt" en caso de imagenes.

Mark

Define un texto resaltado, normalmente se muestra con fondo amarillo
